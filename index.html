Kalkulator Segitiga

Ini adalah sebuah website perhitungan keliling dan luas kalkulator dengan menggunakan html, css, dan javascript.

Keliling Segitiga

Rumus keliling segitiga adalah

K = S1 + S2 + S3

Dimana

K = Keliling

S1 = Sisi A

S2 = Sisi B

S3 = Sisi C

Luas Segitiga

Rumus kluas segitiga adalah

L = 1/2 x a x t

Dimana

L = Luas

a = Panjang alas

t = Tinggi
